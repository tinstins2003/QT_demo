
# How to Run the API

1. **Navigate to the project directory**  
   ```bash
   cd /to-do-list-api
   ```

2. **Install Yarn globally**  
   ```bash
   npm install --global yarn
   ```

3. **Install project dependencies**  
   ```bash
   yarn
   ```

4. **Start the API**  
   ```bash
   yarn start
   ```
